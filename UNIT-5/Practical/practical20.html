Portfolio Deployment Guide

A step-by-step walkthrough for Git and hosting.

Phase 1: Git Initialization

Run these commands in your project terminal to start tracking your work:

# Initialize the repository
git init

# Stage all your files
git add .

# Create your first commit
git commit -m "Initial commit: Portfolio setup"

Phase 2: Connecting to GitHub

  1. Create a new repository on GitHub.
  2. Link your local folder and push:
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/your-repo.git
git push -u origin main

Phase 3: Deployment Options

Option A: GitHub Pages

  • Go to Settings > Pages in your GitHub repo.
  • Select Deploy from a branch.
  • Choose main and click Save.

Option B: Netlify

  • Log in to Netlify and select "Import from Git".
  • Connect your GitHub account and choose the repository.
  • Click Deploy site.
Quick Tip: Ensure your main entry file is named index.html and is located in the root directory!
